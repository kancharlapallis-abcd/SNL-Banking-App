## 🚀 How to Get Started

### Step 1: Setup Database (1 minute)
```bash
psql -U postgres
CREATE DATABASE snl_banking;
\q
```

### Step 2: Start Backend (Terminal 1)
```bash
cd SNLBanking/backend
mvn clean install
mvn spring-boot:run
# Backend runs at http://localhost:8080
```

### Step 3: Start Frontend (Terminal 2)
```bash
cd SNLBanking/frontend
npm install
npm start
# Frontend opens at http://localhost:3000
```

### Done! 🎉
Your complete banking application is now running!

```bash

# Run
java -jar target/snl-banking-app-1.0.0-SNAPSHOT.jar

# Or Docker
```

```bash
# Production build
npm run build

# Deploy to Netlify/Vercel
netlify deploy --prod --dir=build
```

✅ **Database Deployment**
```bash
# Backup
pg_dump -U postgres snl_banking > backup.sql

# Restore
psql -U postgres snl_banking < backup.sql
```
